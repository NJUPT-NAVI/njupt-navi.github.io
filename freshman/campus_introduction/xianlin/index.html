食堂介绍

Section titled “食堂介绍”

  1. 南一

    • 定位：最早的三层老食堂，梅兰一条街末端，靠近教学楼（附近：蜜雪冰城、二手书店）
    • 楼层：
      • 1F 甜品/饮料 + 快餐；
      • 2F（外侧）新疆风味；
      • 3F 台球馆 / 家常菜 / 火锅
    • 高峰：工作日 11:40–12:10，18:00 前后

    小贴士

    • 3F 桌子上有免费插座

  2. 南二

    • 定位：三层，靠近图书馆与晨跑起点
    • 楼层：
      • 1F（兰苑侧）小卖部 / （菊苑侧）烘焙店；
      • 2F 称重自选菜；
      • 3F 健身器材 / 招聘活动
    • 高峰：晨跑后、11:45、18:00

  3. 南三

    • 定位：最新翻新（2025 暑期）两层，含 蜜雪冰城 / 麦当劳 / 小吃街；外有大型超市
    • 楼层：
      • 1F 快餐 + 烘焙 + 品牌档口；
      • 2F 甜品店 + 咖啡店
      • 2F 侧面 小卖部 + 麦当劳 + 小吃街延伸档
    • 高峰：晨跑日 7:20–7:40；午 11:50；晚 18:10 左右

宿舍相关问题

Section titled “宿舍相关问题”
宿舍是如何确定下来的？
Section titled “宿舍是如何确定下来的？”

宿舍是开学前由导员抽签决定的。通常一个寝室尽可能分配同一个专业的同学。

宿舍可以换吗？
Section titled “宿舍可以换吗？”

目前几乎不行，但身高较高的男生可以申请加长床。 申请加长床安置住宿的新生须同时符合以下2个条件：

  • 一是身高1.88米及以上（以体检信息为准）的仙林校区男生;
  • 二是本人知晓不与本班同学共同住宿，而是与其他班级和学院的同性别超高学生混宿并且愿意服从这样安排。
